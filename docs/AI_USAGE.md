# AI Usage

## Tool
- Claude (Anthropic) — 스캐폴딩, API 구현, 테스트, 문서 작성에 활용

## AI가 한 것
- 요구사항 분해 및 패키지 구조 설계
- 엔티티, DTO, 서비스, 컨트롤러 초안 생성
- 예외 처리 및 시드 데이터 작성
- 단위 테스트 시나리오 작성
- 문서 초안 작성

## 내가 검토한 것
- API 응답 필드와 상태 코드가 명세와 일치하는지
- 현재 기수 기준 조회 로직이 의도대로 동작하는지
- API 22의 `sessionId` 처리 방식 (명세가 모호해서 세션 존재 확인 후 기수 전체 요약 반환으로 결정)
- 실서비스 배포 전 검토 필요: DB 교체, 인증 체계, 로깅/모니터링, 트랜잭션 경계

## 검증
- `./gradlew test` — 전체 테스트 통과 확인
